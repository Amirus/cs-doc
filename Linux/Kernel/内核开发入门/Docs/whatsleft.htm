后面该做些什么？

后面该怎么样开发内核就完全取决于你了。下面你该考虑编写的程序应该是一个内存管理器。内存管理器让你能按照需要动态地分配/释放内存。有了它，你就能使用像链表、二叉树等更加复杂的数据结构。这样，存储和操纵数据将变得十分高效。同时，它也是一种防止应用程序写内核页的保护措施。

你也许应该编写一个VGA显卡的驱动程序。因为它能使你在内核中使用更多的显示模式，以提供更多更高的分辨率和图形选项，例如按钮、图片。如果你想做得更深入，你最后应该专注于VESA的驱动。VESA能提供更多的颜色和更高的分辨率。

你也许还应该编写一个允许你按需加载/卸载内核模块的设备接口。在内核中添加对文件系统的支持，这样，你就能使用磁盘上文件和应用程序了。

你也可以添加多任务支持并设计自己的调度算法。调度算法能使你认为优先的任务拥有更高的优先级和更长的CPU时间来运行。注意，多任务系统需要一个内存管理器来给不同的任务分配不同的内存空间。
