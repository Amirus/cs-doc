VIM

VIM是一种有模式(modal)编辑器, 跟平常的Notepad之类有很大区 别. 起初使用起来很不习惯, 但是熟练之后你会发现有模式的编辑器 好处非常多。

很多人喜欢比较 Emacs 和 VIM 哪一个最好。其实你没有办法说 哪一个最“好”。你只能说哪一个最强大。最强大的编辑器，毫无 疑问是 Emacs，但是它不是在所有情况下都是最好的。

虽然 VIM 没有 Emacs 那么强大，用户那么多。但是它可以号称是世 界第二的编辑器。因为除了 Emacs 真的没有哪一个编辑器有如此强大 的能力和简单一致的操作方式。我很喜欢用它修改系统配置文件，因 为它启动迅速，而且对几乎所有文件类型都有语法加亮。

这里有一本很好的讲述 VIM 的书: Vi Improved。你有空可以看看。

VIM 的作者 Bram Moolenaar 写的一篇文章“Seven habits of effective text editing”，他告诉你怎样高效的编辑文本。
